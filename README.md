# Leave Management System (FastAPI)

## About this project
This is a simple backend API built using FastAPI to manage employee leave data.  
It handles employee details, leave requests, and leave balances using a JSON file instead of a database.

The idea was to keep it lightweight, easy to run, and still cover all the core backend logic like validation, status handling, and data management.

---

## Tech used
- Python (3.10+)
- FastAPI
- Uvicorn
- JSON file as database
